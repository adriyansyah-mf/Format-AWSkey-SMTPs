# Format-AWSkey-SMTPs
Simple Tools Result File Formating (AWS Key / SMTPs)
- AWSkey Format to >> Aws_key|Aws_sec|Aws_reg
- SMTPs  Format to >> mailhost|port|mailuser|mailpass

- [x] Features:
  - With Threads
  - Easy-To-Use
