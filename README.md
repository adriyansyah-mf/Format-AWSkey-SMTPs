# Format-AWSkey-SMTPs
Simple Tools Result File Formating (AWS Key / SMTPs)
- [x] AWSkey Format to >> Aws_key|Aws_sec|Aws_reg
- [x] SMTPs  Format to >> mailhost|port|mailuser|mailpass

- Features:
  - [x] With Threads
  - [x] Easy-To-Use
